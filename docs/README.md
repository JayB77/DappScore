# Welcome to DappScore

## What is DappScore?

**DappScore** is a community-driven platform for discovering and evaluating blockchain projects across 50+ chains. We combine community voting with automated on-chain security signals to help crypto users separate legitimate projects from scams.

{% hint style="success" %}
**Our Mission**: Give every crypto user the tools to make informed decisions — combining the wisdom of the crowd with real on-chain data.
{% endhint %}

***

## 🚨 The Problem

The crypto space has a trust problem:

| Issue                        | Impact                                    |
| ---------------------------- | ----------------------------------------- |
| 💀 **Scams everywhere**      | Rug pulls steal billions yearly           |
| 🏃 **No accountability**     | Bad actors disappear without consequences |
| 🔮 **Information asymmetry** | Users can't properly evaluate projects    |
| 💸 **Paid promotions**       | Many "reviews" are advertisements         |

## 🛡️ Security Signals

Every project page automatically runs 9 on-chain checks:

| Signal                  | What It Detects                               |
| ----------------------- | --------------------------------------------- |
| 🍯 Honeypot Detector    | Tokens you can buy but can't sell             |
| 💧 DEX Liquidity        | Real trading activity and liquidity depth     |
| 🔒 Liquidity Lock       | Whether LP tokens are locked and for how long |
| 🕵️ Deployer History    | Serial ruggers reusing the same wallet        |
| 📊 Token Distribution   | Whale concentration and insider control       |
| 🏅 Audit Badges         | Verified security audit records (20 firms)    |
| 👥 Social Proof         | Discord member count and community activity   |
| 🐋 Whale Tracker        | Large transfers in the last 24 hours          |
| 🔍 Contract Fingerprint | Source verification, proxy detection          |
