## Reporting a Scam Project

If you've found a project on DappScore you believe is a scam:

1. Navigate to the project's page
2. Click **"Report Project"** at the bottom of the page
3. Provide evidence (links, screenshots, contract analysis)
4. Submit — the report is reviewed by the community and admins

Confirmed reports earn you **100–500 $SCORE** in rewards. See [Scam Reporting](../platform/scam-reporting.md) for details.

***

## Reporting a Security Vulnerability

If you've discovered a vulnerability in the DappScore platform or smart contracts:

{% hint style="danger" %}
**Do NOT disclose vulnerabilities publicly.** Please contact us through official channels first to allow for a responsible disclosure process.
{% endhint %}

1. Contact via official channels (Discord/Telegram when available)
2. Provide a description of the vulnerability and potential impact
3. Do not exploit it or share it publicly
4. We aim to acknowledge reports within 48 hours
