# Submitting Projects

Learn how to submit a project to DappScore for community evaluation.

## Requirements

- Connected Web3 wallet
- Project information ready
- USDC for premium listing (optional)

## Submission Steps

### Step 1: General Information

**Required Fields:**
- **Project Name** - The full name of your project
- **Token Symbol** - The ticker symbol (e.g., BTC, ETH)
- **Category** - DeFi, Gaming, AI, NFT, etc.
- **Description** - Detailed explanation of your project

**Optional Fields:**
- Your name and email (for contact)
- Ownership verification (if you're a team member)

### Step 2: Project Details

- **Blockchain** - Primary chain (Base, Ethereum, etc.)
- **Project Stage** - Development status
  - Concept / Idea
  - In Development
  - Testnet Live
  - Mainnet Beta
  - Mainnet Live
  - Fully Launched
- **Contract Addresses** - Add multiple if deployed on multiple chains
- **Token Sale Details** (if applicable)
  - Sale type (ICO, IEO, IDO, Fair Launch, Presale)
  - Hard cap and soft cap
  - Token price
  - Start and end dates

### Step 3: Links & Resources

Add all relevant links:
- Website
- Whitepaper / Pitch Deck
- Twitter, Telegram, Discord
- LinkedIn, Reddit, Medium
- GitHub, YouTube
- Facebook, TikTok, Instagram

### Step 4: Team Information

Add team members:
- Full name
- Role / Position
- Short bio
- Photo URL
- LinkedIn and Twitter profiles

### Step 5: Listing Type

Choose your listing option:

| Option | Price | Benefits |
|--------|-------|----------|
| Free | $0 | Standard listing, community voting |
| Premium | 100 USDC | Featured placement for 7 days |

## Ownership Verification

If you're a team member and want to claim ownership:

1. Check "I am a team member of this project"
2. Click "Sign Message to Verify"
3. Sign the verification message in your wallet
4. Ownership badge will be displayed

Benefits of verified ownership:
- Edit your project listing anytime
- Respond to community feedback
- Access analytics (coming soon)

## Duplicate Detection

The platform automatically checks for:
- Existing projects with same name
- Matching contract addresses

If a duplicate is detected, you'll see a warning but can still submit if it's a different project.

## After Submission

Once submitted:
1. Your project appears in the directory
2. Community can vote and comment
3. Trust score develops over time
4. (Premium) Featured for 7 days at top

## Editing Your Listing

If you verified ownership, you can edit:
1. Go to your project page
2. Click "Edit Project" button
3. Sign verification message
4. Update information
5. Save changes
